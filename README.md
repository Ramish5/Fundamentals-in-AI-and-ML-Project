Project Overview:
 -Goal:To create a simple, smart assistant that helps people in my community manage "Digital Overload." It takes messy, unorganized text and tells you if you need to          act now or if you can relax.
 How it works:
   -Input: The user feeds a message or email into the system.
   -Processing: The AI scans the text for "Features" (keywords) that suggest urgency or routine updates.
   -Classification: The model calculates a score to decide if the message is Urgent, Routine, or Neutral.
   -Explainability: The AI doesn't just give a label; it lists the words that triggered the decision so the human stays in control.
Where does it get data from:
 -Real-World Observation:
    -The "Urgent" and "Routine" word lists aren't random. I got this info by looking at my own inbox from the last month.
         Examples: I noticed that almost every message I missed had the word "deadline" or "required." I also saw that messages I ignored usually had words like  "newsletter" or "potluck."
